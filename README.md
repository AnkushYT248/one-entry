## Run Locally

Clone the project

```bash
  git clone https://github.com/AnkushYT248/one-entry
```

Go to the project directory

```bash
  cd one entry
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
### or with yarn
```
yarn install
```
